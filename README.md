# Applicant Tracking System (ATS)

## Overview
The Applicant Tracking System (ATS) is a web-based recruitment management platform developed to streamline and automate the hiring process for organizations. The system enables administrators to manage job postings, track applications, schedule interviews, and manage candidate data efficiently.

This project was developed as a Professinal level project using PHP, MySQL, JavaScript, and PHPMailer with a focus on improving recruitment workflow and reducing manual HR operations.

---

## Key Features

### Candidate Module
- User Registration & Secure Login
- Candidate Profile Management
- Resume Upload System
- Apply for Available Jobs
- Password Recovery & Email Verification

### Admin Module
- Admin Dashboard
- Add, Edit, Delete, and Restore Job Postings
- View Candidate Applications
- Interview Scheduling System
- Generate Offer Letters in PDF Format
- AI-Based Resume Screening

### Super Admin Module
- User Management System
- Admin Monitoring & Control
- System-Level Dashboard
- Create Admin Account
