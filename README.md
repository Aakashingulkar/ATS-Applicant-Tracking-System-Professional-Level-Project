# Applicant Tracking System (ATS)

## Project Overview
This project is an Applicant Tracking System developed using PHP, MySQL, and PHPMailer.

## Features
- User Registration/Login
- Job Posting
- Resume Upload
- Interview Scheduling
- Email Verification
- Admin Dashboard
- Super Admin Dashboard for User Management
- AI Resume Screening
- Generate Offer Letter using PDF generator for PHP

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- PHPMailer
